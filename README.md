# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
-リモートリポジットは専用のサーバー上で複数人で共有するためのモノなのに対し、ローカルリポジットは自身で利用するために手元に置いているリポジットであるという違いがある


## プッシュとマージの違いは何でしょうか？
-プッシュがリモートリポジットとローカルリポジット間で変更を反映するのに使われているのに対して、マージはリモート内あるいはローカル内で変更を反映するのに使われている
